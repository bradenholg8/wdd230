Chamber of Commerce Website Project

File Naming Conventions

  • Files: index.html, styles.css.

Site Name

Site Name: Ogden-Weber Chamber of Commerce

Site Purpose

The purpose of this site is:

  • To provide local businesses with information and networking opportunities.
  • To promote community events and local attractions.

Scenarios

Users may have the following questions when visiting the Chamber site:

  • What events will the Chamber be hosting this month?
  • How can I join the Chamber and become a member?
  • Where can I find information about local businesses in the area?

Color Schema

The color scheme for the website includes the following colors:

  • #000050 - Used for the header background and navigation bar.
  • #3498db - Used for buttons and links.

Typography

The following font will be used throughout the website:

  • Roboto - Used for the body text and headings.
